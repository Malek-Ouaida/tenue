# Tenue — Product Requirements

Tenue is a visual-first social platform.

Core pillars:
- Identity
- Expression
- Discovery
- Social graph
- Commerce (later)

---

## Functional Requirements

- User accounts
- Profiles
- Media posts
- Feed (chronological first)
- Likes, saves, follows
- Uploads (images first)
- Pagination everywhere

---

## Non-Functional Requirements

- Deterministic APIs
- Scalable data model
- Stateless API
- Mobile-first UX
- Observability-ready

---

## Out of Scope (for now)

- Realtime chat
- Video streaming
- Recommendation ML
- Payments

---
